# Kaggle Data Cleaning

This repository contains my work, notes, exercises, and certification from the Kaggle Data Cleaning course.

##  About

Data cleaning is one of the most important steps in the data analytics workflow. This course focused on identifying, fixing, and preparing messy datasets for analysis.

##  Topics Covered

### Missing Values

* Detecting missing data
* Dropping and filling null values
* Imputation techniques

### Scaling and Normalization

* Min-Max Scaling
* Data Normalization
* Feature transformation

### Parsing Dates

* Converting text into datetime objects
* Extracting day, month, and year information
* Handling inconsistent date formats

### Character Encodings

* Understanding UTF-8 and other encodings
* Fixing encoding-related errors
* Reading files with different encodings

### Inconsistent Data Entry

* Text preprocessing
* String cleaning
* Fuzzy matching using FuzzyWuzzy

##  Technologies Used

* SciPy
* Seaborn
* FuzzyWuzzy
